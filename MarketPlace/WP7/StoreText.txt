Media Browser is a way of browsing your own media collection, whether it be movies, tv or music, Media Browser will allow you to listen to them on your Windows Phone or play them on another compatible Media Browser client.

****IMPORTANT INFORMATION****
In order to use Media Browser, you need to have installed the Media Browser server from http://www.mediabrowser3.com/download and have it configured.

Features:
- Browse your own media collection
- Remote control other Media Browser players (like Media Browser Classic)
- Pin your favourite collections to the start menu

Trial limitations:
- Only able to remote control the Web Interface
